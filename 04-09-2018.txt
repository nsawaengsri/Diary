หลังจากที่ผมได้เริ่มกลับมาศึกษาส่วนที่เพื่อนเขียนไว้
ทำให้ผมพอรู้คร่าวๆ เพิ่มเติมขึ้นมายิ่งขึ้น
เริ่มตั้งแต่ เรื่องของ webserver โดยนิยมจะเป็น server แบบ TCP PORT 80
โดยจะให้ client เข้าร้องขอด้วย web browser Protocal http 
จากนั้นส่วนของ Bord 
จะมีการเรียก library ที่จำเป็น สำหรับ Client server 
ส่วนของการ เชื่อมต่อ wifi 
ส่วนของฟังชันหลัก มีการเรียกใช่ ฟังก์ชันรอง

ต่อไป ผมจะต้องเจาะลึกลงไปอีก เกี่ยวกับการทำงานของ CODE โดยที่ผมจะต้องสามารถ
เข้าใจได้ในทุกบรรทัด ว่าทำอะไร อย่างไร
อาจจะเริ่มจาก คิดว่า ฟังก์ชันที่สร้างมา เขียนมาเพื่ออะไร ผลลัพธ์อะไรที่จะได้จาก Function แล้วมันจะต้องไปไหนต่อ 
ผมจะต้องพยายามเข้าใจให้ได้ในทุกขั้นตอน เอาเริ่มต่อได้เลย

ผมจะเริ่มจากงานที่ผมได้ทดลองด้วยคือ PLC VS NAV SOAP SERVIEC
จะทดลอง ด้วยตัวเอง ทำการ รัน แก้ไข ปรับปรุง ให้ครบ FLOW
