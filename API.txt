API and Web Service = communication.
Web Servie นั้นจะอำนวยความสะดวกการสื่อสารระหว่าง machines 2 machines ที่อยู่ต่าง network กัน 
API  = กระทำกับ 2 application ที่แตกต่างกันให้สามารถทำการสื่อสารได้ 
API = คือวิธี การที่ให้ Third-party vendors(บุคคลที่ 3 ) ให้สามารถเขียนโปรแกรมในการติดต่อกับโปรแกรมอื่นได้
Web Service = การออกแบบ Interface ในรูปแบบ xml
มักจะใช้ Protocol HTTP ในการสื่อสาร  
Web Service use SOAP,REST and XML
