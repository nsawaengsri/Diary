วันนี้ก็เป็นเช้าวันใหม่ เป็นวันที่ต้องทำงาน ในสิ่งที่เราคิดว่าสนุก 
เอาหล่ะ ผมก็คิดว่ามันไม่ได้เลวร้ายอะไรเท่าไร มันก็มีความท้าทาย
มีความสนุกให้ได้ตะลุยต่อ ความไม่สนุกน่าเบื่อ อาจจะเป็นเพราะ
ขาดเพื่อนร่วมงาน ขาดคนที่จะได้แชร์ปัญหา หรือรร่วมแก้ไขปัญหาด้วยกัน
	สิ่งที่ผมต้องการอาจจะยากไปเสียหน่อย อาจจะเป็นปัจจัยภายนอกที่เราคุมไม่ได้
สิ่งที่ผมทำได้ในตอนนี้ คือคุยกับความคิดของตนเอง ลับคมความคิดของตัวเอง ผ่านการเล่า การเขียนมันออกมา
	เวลาผ่านไป ถึง 10 โมงกว่า การดำเนินการประติดประต่อ จากเมื่อวาน ยังไม่ไหลลื่นเลย มีเรื่องนิดหน่อย ในการขัดจังหวะ
ผมจะเริ่ม นิ่ง มองดู เงียบ และเขียนจุดที่ยืนอยู่ จุดที่จะไปต่อ 
	เอาหล่ะ ตอนนี้ผมยังคงต้องแกะ Code อีกสักประมาณ 2 ชม จากเมื่อวานพอจะเข้าใจยิ่งขึ้นกว่าเดิม ว่า Code มีการทำงานอย่างไร 
ผมจะเขียนสิ่งที่เข้าใจจากเมื่อวาน เพื่อเป็นจุดเชื่อมต่อละกัน 
	เมื่อวาน ผมศึกษา Flow งาน เป็น TASK ของการทดลอง nav กับ pcl โดยสื่อสารกันแลกเปลี่ยนข้อมูลผ่าน
Web Service ตัว ESP8266 มี Service ในตัวมันเอง ทาง NAV สามารถสื่อสารได้ผ่าน ip ของ ESP โดยส่งข้อมูลไปในแบบ REST API รูปแบบของ json
ซึ่งแน่น่อน จะเป็น String ทั้งหมด 
	ส่วนของ ESP8266 อย่างที่รู้ในการทำงานของ BORD 
1 มีการเรียกใช่่ Library ที่สำคัญ อาทิ WIFI, CLIENT SERVER , JSON 
2 การ Define variable ที่จำเป็น
3 ฟังชัน setup 
	ในฟังชันจะมีการเรียกใช้ครั้งเดียว
	ส่วนใหญ่ที่เห็นตรงนี้จะใช้เขียนเกี่ยวกับ การชื่อต่อกับ WIFI 
4 ฟังก์ชัน Loop ใช้สำหรับการทำงานของ Brod จะเป็นส่วนที่เกี่ยงกับ Business process ที่เรากำหนด 
5 เขียนฟังก์ชันรอง ตาม Algorithm

	ตอนนี้ก็ได้ทดทวนภาพใหญ่จากเมื่อวาน แต่ยังไม่เพียงพอสำหรับผม ผมยังไม่เข้าใจทั้งหมด ของ Code ข้างใน
มันมีบางจุด ที่ทำให้ผมมีคำถามว่า มันคืออะไร สร้างมาทำไม ทำไมต้องสร้าง สร้างแล้วใช้ตรงไหน ส่งผลกับตรงไหน  
วันนี้แหละ ผมตั้งเป้าว่าจะตอบให้ได้ 
	ผมคิดถึงผลลัพธ์ที่คาดหวังของตัวเอง ผมสามารถเข้าใจ Code ทั้งหมดได้ จากนั้น ผมจะเขียน FLOW ได้
เพื่อกันลืม เพื่อสื่อสารกับผู้อื่น เพื่อทดสอบตัวเองว่า เข้าใจจริงไหม จะได้นำไปปรับปรุงได้  ไม่มีอะไร เพอร์เฟ็ก แต่ต้องทำ 100 % ที่รู้ให้สนุกสุดมันส์ไปเลย 
	สนุกเริ่มรู้สนุกขึ้นมา เมื่อได้เขียนมันออกมา ผ่านความคิดคำพูด เอาหล่ะ 
ทบทวนเป้าหมายของ Project นี้กันหน่อย จะได้ไปต่อ อย่างมีเป้าหมาย Project นี้สร้างเพื่อ เติมเต็มกับ nav เพื่อรองรับ อุตสาหกรรมแบบ 4.0 
 Project ที่เดินทางผ่านมาแล้ว
